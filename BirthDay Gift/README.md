# Photography Studio Website

## Description
This is a static website for a photography studio. It showcases the studio's portfolio, services, and contact information. The website is built using HTML, CSS, and JavaScript.

## Features
- Hero section with stunning images
- Portfolio gallery
- Responsive design
- Interactive elements powered by JavaScript

## File Structure
- `final.html`: Main HTML file for the website
- `script.js`: JavaScript file for interactivity
- `style.css`: CSS file for styling
- `Asset/`: Folder containing hero images (1 Hero.avif, 2 Hero.avif)
- `Portfolio/`: Folder containing portfolio page (collection.html)

## How to View the Website
1. Open `final.html` in your web browser.
2. Navigate through the sections to explore the studio's work.

## Technologies Used
- HTML5
- CSS3
- JavaScript
